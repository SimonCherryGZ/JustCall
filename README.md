
JustCall
===================================  
JustCall是一款定时拨打电话的应用。

已经实现的内容有：
- 读取手机联系人列表。
- 联系人列表首字母快速查找。
- 调用全局定时器实现定时拨打电话。
- 定时模式分为即时、定时和随机时间。
- 设置常用联系人，实现快速拨打。

应用平台 
-----------------------------------  
  Android
